# Streamlit Learning Project

This repository contains examples and demonstrations for learning Streamlit, a powerful Python library for creating web applications.

## Setup Instructions

1. Install the required dependencies:
   ```bash
   pip install -r requirements.txt
   ```

2. Run the Streamlit app:
   ```bash
   streamlit run app.py
   ```
